Research Tips

Helpful Tips

Use Citations:

  • Type of record
  • Place
  • Date of Event
  • Name of Individual
  • Reference
  • Where the date was accessed
  • Date the record was accessed

Search the U.S Census

  • The census tells you the name and given place of person
  • Search each Census one by one
  • Start with the most recent census
  • Start broad with the basic information
  • Study the image and find detailed information

Church and Death Records

  • Churches have kept excellent records,
    especially if you are searching worldwide
  • These records contain baptisms, marriages,
    divorces, deaths and burials
  • Death records can give information such as
    military unit, profession or achievements
  • Obituaries can sometimes tell ones life story

Search Different Spellings of Surnames:

  • Say the surname outloud
  • Add a silent "H"
  • Look for other silent letters
  • Try different vowels
  • Add or remove an ending "S"
  • Look for commonly misread letters
  • Consider name changes
Example:

My great grandfathers last name was DiGirolamo,
his marriage certificate spelt the name
"DeGrolamo". From that I was able to find
the passenger list when he immigrated to
America from Sicily.
